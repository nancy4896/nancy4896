엔터테인먼트 디자인이란?

엔터테인먼트 디자인 사람들에게 재미, 즐거움을 주는 디자인입니다.
즉 사용자들의 경험을 분석을 토대로 해결방안을 찾아 재미를 주는 디자인을 기획하고 콘텐츠를 제작합니다. 그래서 영상, 웹 디자인, UX디자인, 기획 등의 여러 방면을 익힙니다.

자신이 생각하는 '좋은디자인'이란?

사용자가 불편함 없이 만족하는 디자인이 좋은 디자인이라고 생각합니다.
디자이너의 머릿속에서 나오는 것이 아닌 사용자의 경험들을 분석해 디자인해주는 것입니다.
사용자의 심리 분석, Culture Code 등 여러 가지들을 고려해 적재적소에 필요한 디자인을 해주는 것이 좋은 디자인이라고 생각합니다.
